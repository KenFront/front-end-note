# Linux cmd

### 查看當前目錄

```bash
ls

ls -a //all file

ls *.js //filter
```

### 前往其他目錄

```bash
cd
```

### 顯示當前目錄

```bash
pwd
```

### 創建新目錄

```bash
mkdir hello
```

### 創建新檔案

```bash
touch hello.js
```

### 複製檔案

```bash
cp ./hello.js ./world.js
```

### 刪除檔案

```bash
rm hello.js
```

### 刪除目錄

```bash
rm -rf hello
```


### 刪除目錄

```bash
rm -rf hello
```

### 移動檔案

```bash
mv hello.js ../world.js
```


### 查看檔案全部內容

```bash
cat hello.js
```

### 查看檔案最後部分內容

```bash
tail hello.js
```
